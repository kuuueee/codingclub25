Home Tutorials Showcase

Let's start first by cleaning up your homepage!

The main function of your home page will be to link to each of your different pages, hence a website:

The top left Home button will take you to the Mr.C's homepage, nothing really cool happens there, but it did link you to here!

The following guide will present you with an opportunity to start creating your website as it branches from your homepage

Step1.

There is a file in your project directory called html/template.html

Make a copy of this file, and rename it to some name: enter that name here:

Just the name, not the file extension


Step2.

Add a Navigation bar to the top of your website, you will reuse this to begin every new page you create.

Copy this code into your index.html

				<div class="topnav">
					<a href="#">Home</a>
					<a href="html/template.html">Template</a>
				</div>

Step 3.

Configure the code on

template.html

Notice the difference in paths for where documents are located relative to each other.

				<div class="topnav">
					<a href="../index.html">Home</a>
					<a href="#">Template</a>
				</div>

Step 4.

Notice the div class: we should steal some style. What we have now kinda looks like junk.

More code to copy into your css/home.css file

					.topnav {
					  background-color: #333;
					  overflow: hidden;
					  border-style: solid; 
					  border-color: #ddd;
					  text-align: center;
					  color: red;
					}

					/* Change the color of links on hover */
					.topnav a:hover {
					  background-color: #ddd;
					  color: black;
					}


					/* Style the links inside the navigation bar */
					.topnav a {
					  float: left;
					  color: #f2f2f2;
					  text-align: center;
					  padding: 14px 16px;
					  text-decoration: none;
					  font-size: 17px;
					}


					/* Add a color to the active/current link */
					.topnav a.active {
					  background-color: #ac32bf;
					  color: white;
					}

Play with the colours, fonts and padding to aquire your navigation pane. It should look the same from both pages.

Step 5.

Make sure the links take you back and forth between both pages, and that all the styles are applied to both pages.

If your linking doesn't work:

Look at each page and make sure the links go to where they say they are going.

If your style isn't applied, make sure that your html page is referencing the style properly.
